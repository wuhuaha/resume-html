# Resume HTML

This repository contains an AI-assisted online resume and profile page.

The active application lives in `ai-profile-page/`. It includes a Vue 3 frontend,
a FastAPI backend, Markdown-driven profile content, admin editing tools, AI Q&A,
voice features, and resume export configuration.

## Project

- App: `ai-profile-page/`
- Frontend: `ai-profile-page/frontend/`
- Backend: `ai-profile-page/backend/`
- Content: `ai-profile-page/content/`
- Documentation: `ai-profile-page/README.md`

## Local Development

See `ai-profile-page/README.md` for setup, configuration, screenshots, and
deployment notes.
